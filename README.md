# OneRule4All
OneRule4All is a simple C# program that creates a new user account, adds the user to the Administrators group, and enables RDP and PSRemoting for the new user.

## Getting Started
### Prerequisites
To run this program, you will need:
* Visual Studio or another C# compiler
* Administrative privileges on the machine where you want to create the new user account

### Installation
1. Clone the repository to your local machine using `git clone https://github.com/your_username/OneRule4All.git`
2. Open the solution file `OneRule4All.sln` in Visual Studio or your preferred `C#` compiler
3. Build the solution
4. Run the program

## Usage
The program will create a new user account, add the user to the Administrators group, and enable RDP and PSRemoting for the new user. You can customize the username and password in the code if desired.
